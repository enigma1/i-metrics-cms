Here you see a list of text pages to which you can assign the custom boxes.
Do the following to assign the pages.

  • Using he checkboxes on the left select the text pages
  • Hit Update

Once you finish with the assignments, you can return to the main assignment page by hitting the back button. From there you can set the title and content of the box for each text page.

Also note, You can assign the same page multiple times. A separate entry will be created.
Each entry refers to a separate box.
Boxes can be ordered using the Display Order input box.
