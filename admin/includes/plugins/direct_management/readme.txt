<p>The Direct Management plugin offers edit capability of the site's content displayed on the web-front. An additional plugin entry is visible under the site content section once this plugin is installed.</p>
<p>To activate it goto Site Content then select the Direct Management option.</p>
<p>Once installed an administration link will create a special session for the administrator to edit directly text pages and collections on the main site.</p>
<p>The front administration management is further secured by an automatically generated parameter key in conjuction with back-end administrator's IP verification.</p>
